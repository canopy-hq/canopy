# Superagent

Monorepo for the Superagent desktop app.

## Commands

From root using `--filter`:

```bash
bun run --filter desktop tauri dev     # Start dev mode
bun run --filter desktop tauri build   # Build the app
bun run --filter desktop test          # Run tests
bun run --filter desktop dev           # Start Vite dev server only
bun run --filter desktop build         # Build frontend only
```
