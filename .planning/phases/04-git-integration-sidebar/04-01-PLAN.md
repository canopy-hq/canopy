**2. Create `src-tauri/src/git.rs` with these types and commands:**

Serializable types:
```rust
#[derive(Serialize, Clone)]
pub struct BranchInfo {
    pub name: String,
    pub is_head: bool,
    pub ahead: usize,
    pub behind: usize,
}

#[derive(Serialize, Clone)]
pub struct WorktreeInfo {
    pub name: String,
    pub path: String,
}

#[derive(Serialize, Clone)]
pub struct RepoInfo {
    pub path: String,
    pub name: String,
    pub branches: Vec<BranchInfo>,
    pub worktrees: Vec<WorktreeInfo>,
}
```

Tauri commands (all `pub`, all return `Result<T, String>`):

- `import_repo(path: String) -> Result<RepoInfo, String>`: Opens repo via `Repository::open(&path)`. Extracts repo name from path (last component). Lists local branches via `repo.branches(Some(BranchType::Local))`. For each branch: get name, is_head, ahead/behind via `repo.graph_ahead_behind(local_oid, upstream_oid)` -- wrap `branch.upstream()` in match, return (0,0) if no upstream (Pitfall 2). Lists worktrees via `repo.worktrees()` + `repo.find_worktree(name)` -- skip invalid ones via `wt.validate()`. Returns `RepoInfo`.

- `list_branches(repo_path: String) -> Result<Vec<BranchInfo>, String>`: Same branch enumeration as import_repo (extract to helper fn `enumerate_branches(&Repository)`).

- `create_branch(repo_path: String, name: String, base: String) -> Result<BranchInfo, String>`: Opens repo, finds base branch via `repo.find_branch(&base, BranchType::Local)`, peels to commit, calls `repo.branch(&name, &commit, false)`. Returns new BranchInfo with ahead=0, behind=0.

- `delete_branch(repo_path: String, name: String) -> Result<(), String>`: Opens repo, finds branch, checks `is_head()` -- if true, return Err("Cannot delete the currently checked-out branch.") (Pitfall 5). Otherwise calls `branch.delete()`.

- `create_worktree(repo_path: String, name: String, path: String, base_branch: Option<String>) -> Result<WorktreeInfo, String>`: Opens repo. If `base_branch` is Some, find branch, get reference, set `WorktreeAddOptions::reference`. Call `repo.worktree(&name, Path::new(&path), Some(&opts))`. Validate target path doesn't exist before calling (Pitfall 3). Returns WorktreeInfo.

- `remove_worktree(repo_path: String, name: String) -> Result<(), String>`: Opens repo, finds worktree via `repo.find_worktree(&name)`. Checks `wt.is_locked()` -- if locked, return Err("Worktree \"{name}\" is locked and cannot be removed.") (Pitfall 4). Calls `wt.prune(None)` with default prune options.

Helper function (not a command):
```rust
fn enumerate_branches(repo: &Repository) -> Result<Vec<BranchInfo>, String> { ... }
```

**CRITICAL: git2::Repository is NOT Send. Do NOT store it in Tauri state. Open fresh per command.**

**1. Create `src/lib/git.ts` -- typed IPC wrappers:**

```typescript
import { invoke } from '@tauri-apps/api/core';

export interface BranchInfo {
  name: string;
  isHead: boolean;
  ahead: number;
  behind: number;
}

export interface WorktreeInfo {
  name: string;
  path: string;
}

export interface RepoInfo {
  path: string;
  name: string;
  branches: BranchInfo[];
  worktrees: WorktreeInfo[];
}

export function importRepo(path: string): Promise<RepoInfo> {
  return invoke<RepoInfo>('import_repo', { path });
}

export function listBranches(repoPath: string): Promise<BranchInfo[]> {
  return invoke<BranchInfo[]>('list_branches', { repoPath });
}

export function createBranch(repoPath: string, name: string, base: string): Promise<BranchInfo> {
  return invoke<BranchInfo>('create_branch', { repoPath, name, base });
}

export function deleteBranch(repoPath: string, name: string): Promise<void> {
  return invoke<void>('delete_branch', { repoPath, name });
}

export function createWorktree(repoPath: string, name: string, path: string, baseBranch?: string): Promise<WorktreeInfo> {
  return invoke<WorktreeInfo>('create_worktree', { repoPath, name, path, baseBranch: baseBranch ?? null });
}

export function removeWorktree(repoPath: string, name: string): Promise<void> {
  return invoke<void>('remove_worktree', { repoPath, name });
}
```

**2. Create `src/stores/workspace-store.ts` -- Zustand store with immer:**

Follow same pattern as tabs-store.ts (create + immer middleware).

```typescript
import { create } from 'zustand';
import { immer } from 'zustand/middleware/immer';
import type { BranchInfo, WorktreeInfo, RepoInfo } from '../lib/git';
import * as gitApi from '../lib/git';
import { showErrorToast } from '../lib/toast';

export interface Workspace {
  id: string;          // crypto.randomUUID()
  path: string;        // absolute filesystem path
  name: string;        // repo name (last path component)
  branches: BranchInfo[];
  worktrees: WorktreeInfo[];
  expanded: boolean;
}

interface WorkspaceState {
  workspaces: Workspace[];
  sidebarVisible: boolean;
  sidebarWidth: number;        // default 230
  selectedItemId: string | null; // tree selection

  // Sidebar
  toggleSidebar: () => void;
  setSidebarWidth: (width: number) => void;

  // Workspace CRUD
  importRepo: (path: string) => Promise<void>;
  removeRepo: (id: string) => void;
  refreshRepo: (id: string) => Promise<void>;
  toggleExpanded: (id: string) => void;
  setSelectedItem: (id: string | null) => void;

  // Git operations
  createBranch: (workspaceId: string, name: string, base: string) => Promise<void>;
  deleteBranch: (workspaceId: string, name: string) => Promise<void>;
  createWorktree: (workspaceId: string, name: string, path: string, baseBranch?: string) => Promise<void>;
  removeWorktree: (workspaceId: string, name: string) => Promise<void>;
}
```

Store implementation:
- `toggleSidebar`: flips `sidebarVisible`
- `setSidebarWidth`: clamps to 180-400 range
- `importRepo(path)`: calls `gitApi.importRepo(path)`, on success creates Workspace with `id: crypto.randomUUID()`, `expanded: true`, pushes to workspaces array, sets `sidebarVisible: true`. On error, calls `showErrorToast('Import failed', error)`.
- `removeRepo(id)`: filters workspace out of array
- `refreshRepo(id)`: finds workspace by id, calls `gitApi.listBranches(ws.path)`, updates branches array. Also re-lists worktrees via `gitApi.importRepo(ws.path)` and merges worktrees.
- `toggleExpanded(id)`: flips `expanded` for workspace
- `createBranch(workspaceId, name, base)`: finds workspace, calls `gitApi.createBranch(ws.path, name, base)`, then calls `refreshRepo(workspaceId)`. On error, showErrorToast.
- `deleteBranch(workspaceId, name)`: finds workspace, calls `gitApi.deleteBranch(ws.path, name)`, then refreshRepo. On error, showErrorToast.
- `createWorktree(workspaceId, name, path, baseBranch)`: same pattern with `gitApi.createWorktree`
- `removeWorktree(workspaceId, name)`: same pattern with `gitApi.removeWorktree`

Default state: `workspaces: [], sidebarVisible: false, sidebarWidth: 230, selectedItemId: null`
