**2. Create `src/components/WorkspaceTree.tsx`:**

Uses React ARIA `Tree`, `TreeItem`, `TreeItemContent` from `react-aria-components`.

Props: none (reads from workspace store directly).

Structure:
```tsx
<Tree aria-label="Workspaces" selectionMode="single"
      selectedKeys={selectedItemId ? new Set([selectedItemId]) : new Set()}
      onSelectionChange={...}>
  {workspaces.map(ws => (
    <TreeItem key={ws.id} id={ws.id} textValue={ws.name}>
      <TreeItemContent>
        {/* Repo header: name (13px semibold), chevron */}
        {/* Below name: current branch in 11px muted (HEAD branch) */}
      </TreeItemContent>
      {ws.expanded && (
        <>
          {ws.branches.map(b => (
            <TreeItem key={`${ws.id}-branch-${b.name}`} id={`${ws.id}-branch-${b.name}`} textValue={b.name}>
              <TreeItemContent>
                {/* 16px indent, branch icon U+2387 in var(--branch-icon), name 13px */}
                {/* Right-aligned: ahead in var(--git-ahead) "+N", behind in var(--git-behind) "-N" */}
              </TreeItemContent>
            </TreeItem>
          ))}
          {ws.worktrees.map(wt => (
            <TreeItem key={`${ws.id}-wt-${wt.name}`} id={`${ws.id}-wt-${wt.name}`} textValue={wt.name}>
              <TreeItemContent>
                {/* 16px indent, worktree icon U+25C6 in var(--worktree-icon), name 13px */}
              </TreeItemContent>
            </TreeItem>
          ))}
          {/* SIDE-06: "+ New Branch" button */}
          <div style={{ paddingLeft: '16px' }} className="h-7 flex items-center">
            <button className="text-[var(--text-muted)] hover:text-[var(--accent)] text-[13px]"
                    onClick={() => {/* will trigger modal -- wired in Plan 03 */}}>
              + New Branch
            </button>
          </div>
        </>
      )}
    </TreeItem>
  ))}
</Tree>
```

Tree item heights: 28px (`h-7`).
Branch icon: `<span style={{ color: 'var(--branch-icon)' }}>&#x2387;</span>`
Worktree icon: `<span style={{ color: 'var(--worktree-icon)' }}>&#x25C6;</span>`
Ahead/behind format: `{b.ahead > 0 && <span style={{ color: 'var(--git-ahead)' }}>+{b.ahead}</span>}` `{b.behind > 0 && <span style={{ color: 'var(--git-behind)' }}>-{b.behind}</span>}`

Selected state: 2px left border in `var(--accent)`, `var(--bg-tertiary)` background.
Hover state: `var(--bg-tertiary)` background full-width.

Clicking repo header toggles expand via `useWorkspaceStore(s => s.toggleExpanded)`.
