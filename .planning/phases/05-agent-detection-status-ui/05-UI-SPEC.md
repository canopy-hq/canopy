# Phase 5 — UI Design Contract

> Visual and interaction contract for agent detection status indicators, overlay, and toast notifications. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | react-aria-components 1.16.0 (headless) |
| Icon library | Unicode glyphs (established pattern: branch, worktree, chevrons) |
| Font | Menlo, Monaco, "Courier New", monospace (system monospace stack) |

## Color

### Existing Theme Tokens (unchanged)

| Role | CSS Variable | Value (Obsidian default) | Usage |
|------|-------------|-------------------------|-------|
| Dominant (60%) | `--bg-primary` | #0a0a14 | App background, overlay backdrop underlay |
| Secondary (30%) | `--bg-secondary`, `--bg-tertiary` | #12121f, #1a1a2e | Cards, sidebar, tab bar, overlay surface |
| Accent (10%) | `--accent` | #3b82f6 | Focus rings, active tab border (unchanged) |
| Destructive | `--destructive` | #ef4444 | Error toasts only (unchanged) |

### New Agent Status Tokens (added to :root and all 8 themes)

| Token | Value | Usage |
|-------|-------|-------|
| `--agent-running` | #4ade80 | Running status dot fill |
| `--agent-waiting` | #fbbf24 | Waiting status dot fill, pill badge bg, waiting tab tint |
| `--agent-running-pulse` | #22c55e | Pulse animation keyframe target (green-500) |
| `--agent-waiting-glow` | rgba(251, 191, 36, 0.15) | Pane amber border glow background, waiting tab background tint |
| `--agent-waiting-border` | rgba(251, 191, 36, 0.4) | Pane amber border color when agent is waiting |
| `--agent-waiting-inset` | rgba(251, 191, 36, 0.06) | Pane inset shadow color when agent is waiting |

These tokens are IDENTICAL across all 8 themes (same pattern as `--branch-icon`, `--worktree-icon`, `--git-ahead`, `--git-behind` -- defined once in `:root`, not overridden per theme).

Accent reserved for: active tab top border, focus rings, selected overlay row border. Agent status colors are NOT accent -- they are semantic (green = running, amber = waiting).

## StatusDot Spec

| Property | Value |
|----------|-------|
| Size (default) | 8px width x 8px height |
| Size (sidebar collapsed summary) | 6px |
| Size (overlay) | 8px |
| Border radius | 50% (circle) |
| Running color | `var(--agent-running)` |
| Running animation | `pulse-slow` -- opacity oscillates 1.0 to 0.4 over 2s, ease-in-out, infinite |
| Waiting color | `var(--agent-waiting)` |
| Waiting animation | `breathe` -- scale oscillates 1.0 to 1.3 over 2.5s, ease-in-out, infinite |
| Idle | Not rendered (hidden, no DOM element) |

### CSS Keyframes

```css
@keyframes pulse-slow {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.4; }
}

@keyframes breathe {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.3); }
}
```

## Pane Waiting Glow Spec (AGNT-10)

When agent status is `waiting` on a pane:

| Property | Value |
|----------|-------|
| Border | 1px solid `var(--agent-waiting-border)` |
| Box shadow (outer glow) | `0 0 12px var(--agent-waiting-glow)` |
| Box shadow (inset) | `inset 0 0 24px var(--agent-waiting-inset)` |
| Transition | `border-color 0.3s ease, box-shadow 0.3s ease` |

When agent is `running` or `idle`: no border change, no shadow (revert to default pane border).

## Agent Overlay Spec (AGNT-05, AGNT-06)

| Property | Value |
|----------|-------|
| Trigger | Cmd+Shift+O keyboard shortcut (toggle) |
| Dismiss | Esc key, click outside, or Cmd+Shift+O again |
| Position | Centered horizontally and vertically in viewport |
| Width | 520px fixed |
| Max height | 60vh |
| Background | `color-mix(in srgb, var(--bg-secondary) 85%, transparent)` |
| Backdrop | `backdrop-filter: blur(12px)` (frosted glass, per D-13) |
| Border | 1px solid `var(--border)` |
| Border radius | 12px |
| Shadow | `0 8px 32px rgba(0, 0, 0, 0.5)` |
| Overlay backdrop | `rgba(0, 0, 0, 0.3)` (click target for dismiss) |
| Enter animation | opacity 0 to 1 + scale 0.96 to 1.0, 150ms ease-out |
| Exit animation | opacity 1 to 0, 100ms ease-in |

### Overlay Header

| Property | Value |
|----------|-------|
| Padding | 16px 16px 12px 16px |
| Title | "Agent Overview" at 14px semibold, `var(--text-primary)` |
| Counter pill | Right of title, 11px regular, e.g. "2 running" in green + dot separator + "1 waiting" in amber |
| Border bottom | 1px solid `var(--border)` |

### Overlay Body (scrollable)

| Property | Value |
|----------|-------|
| Padding | 8px 0 |
| Overflow | `overflow-y: auto` with scrollbar-width: none |
| Grouping | Rows grouped by workspace name |
| Group header | Workspace name at 11px semibold, `var(--text-muted)`, padding 8px 16px 4px 16px |

### Overlay Row

| Property | Value |
|----------|-------|
| Height | 36px |
| Padding | 0 16px |
| Layout | flex row, items-center, gap 8px |
| Columns | StatusDot (8px) | Agent name (13px, `--text-primary`, flex-shrink-0) | Branch name (13px, `--text-muted`, truncate, flex-1) | Duration (11px, `--text-muted`, tabular-nums, flex-shrink-0) |
| Hover | Background `var(--bg-tertiary)`, border-radius 6px |
| Selected (keyboard) | Background `var(--bg-tertiary)`, left 2px border accent |
| Waiting row tint | Background `var(--agent-waiting-glow)` |
| Click / Enter | Jump to workspace tab + pane, close overlay |
| Transition | Row appear/disappear with opacity fade 200ms |

### Overlay Empty State

| Property | Value |
|----------|-------|
| Layout | Centered vertically and horizontally in body area |
| Heading | "No agents running" at 14px semibold, `var(--text-muted)` |
| Body | "Start an AI agent in any terminal to see it here" at 13px regular, `var(--text-muted)` opacity 0.6 |
| Spacing | 8px gap between heading and body |

### Overlay Duration Format

| Duration | Display |
|----------|---------|
| < 60s | "Ns" (e.g. "42s") |
| 1-59m | "Nm Ns" (e.g. "3m 12s") |
| >= 1h | "Nh Nm" (e.g. "1h 23m") |

Ticking: update every 1s via `setInterval` while overlay is open. Stop interval on close.

### Overlay Keyboard Navigation

| Key | Action |
|-----|--------|
| Arrow Down | Select next row |
| Arrow Up | Select previous row |
| Enter | Jump to selected row's workspace/tab/pane |
| Esc | Close overlay |

## Toast Notification Spec (AGNT-07, AGNT-08, AGNT-09)

| Property | Value |
|----------|-------|
| Position | Bottom-right, 24px from edges (matches existing error toast region) |
| Max visible | 3 toasts |
| Width | 320px |
| Background | `var(--bg-tertiary)` |
| Border | 1px solid `var(--border)` |
| Border radius | 8px |
| Shadow | `0 4px 16px rgba(0, 0, 0, 0.4)` |
| Padding | 12px 16px |
| Enter animation | slide-in from right + fade, 200ms ease-out |
| Exit animation | fade-out + slide-right, 150ms ease-in |

### Toast Content Layout

```
[StatusDot] [Agent name]  [workspace/branch]     [x close]
            [event description]
            [Jump]  [Dismiss]
```

| Element | Style |
|---------|-------|
| Agent name | 13px semibold, `var(--text-primary)` |
| Workspace/branch | 11px regular, `var(--text-muted)`, right-aligned |
| Event description | 11px regular, `var(--text-muted)` |
| "Jump" button | 11px semibold, `var(--accent)` text, no background, hover underline |
| "Dismiss" button | 11px regular, `var(--text-muted)` text, no background, hover underline |
| Close (x) | 10px, `var(--text-muted)`, hover `var(--text-primary)` |

### Toast Types

| Type | Status dot | Description copy | Timeout |
|------|-----------|-----------------|---------|
| agent-waiting | Amber breathing | "{agentName} is waiting for input" | Infinity (persists until dismissed or Jump) |
| agent-complete | Green (static, no pulse) | "{agentName} finished" | 10000ms (auto-dismiss) |

### Toast De-duplication

Only show toasts for events in NON-ACTIVE workspaces (active workspace changes are visible directly). Track `lastToastTime` per ptyId -- suppress duplicate toasts within 5s window.

## Status Bar Agent Summary Spec

| Property | Value |
|----------|-------|
| Location | Right side of status bar, before shortcut hints |
| Format | "{N} working" in green + " {N} waiting" in amber (only show counts > 0) |

| Colors | Running count text: `var(--agent-running)`, Waiting count text: `var(--agent-waiting)` |
| Separator | " " (dot separator between running and waiting, `var(--text-muted)` at 60% opacity) |
