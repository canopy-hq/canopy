**AgentOverlay.tsx:** Create component following CreateModal dialog pattern (plain div overlay + react-aria Dialog, NOT ModalOverlay).

Props: `{ isOpen: boolean; onClose: () => void }`

**Structure (per UI-SPEC):**

1. **Outer backdrop:** Fixed overlay `position: fixed, inset: 0, backgroundColor: 'rgba(0, 0, 0, 0.3)'`, onClick calls onClose

2. **Panel:** Centered, 520px wide, max 60vh tall:
```typescript
style={{
  position: 'fixed',
  top: '50%',
  left: '50%',
  transform: 'translate(-50%, -50%)',
  width: '520px',
  maxHeight: '60vh',
  background: 'color-mix(in srgb, var(--bg-secondary) 85%, transparent)',
  backdropFilter: 'blur(12px)',
  border: '1px solid var(--border)',
  borderRadius: '12px',
  boxShadow: '0 8px 32px rgba(0, 0, 0, 0.5)',
  display: 'flex',
  flexDirection: 'column',
  fontFamily: 'Menlo, Monaco, "Courier New", monospace',
}}
```

3. **Header (16px padding, border-bottom):**
   - "Agent Overview" title: 14px semibold, `var(--text-primary)`
   - Counter pill right of title: `{runningCount} running` in green + dot separator + `{waitingCount} waiting` in amber (11px)
   - Only show non-zero counts

4. **Body (scrollable, overflow-y: auto, scrollbar-width: none):**
   - Group agents by workspace name
   - Group header: workspace name at 11px semibold, `var(--text-muted)`, padding `8px 16px 4px 16px`
   - Agent rows: 36px height, flex row, items-center, gap 8px, padding 0 16px

5. **Agent row layout:**
   - StatusDot (8px)
   - Agent name: 13px, `var(--text-primary)`, flex-shrink-0
   - Branch name: 13px, `var(--text-muted)`, truncate, flex-1
   - Duration: 11px, `var(--text-muted)`, tabular-nums, flex-shrink-0
   - Hover: `var(--bg-tertiary)` bg, border-radius 6px
   - Selected (keyboard): `var(--bg-tertiary)` bg + left 2px border accent
   - Waiting row tint: `var(--agent-waiting-glow)` background

6. **Empty state:** Centered in body:
   - "No agents running" at 14px semibold, `var(--text-muted)`
   - "Start an AI agent in any terminal to see it here" at 13px, `var(--text-muted)` opacity 0.6
   - 8px gap between heading and body

7. **Duration formatting:**
```typescript
function formatDuration(startedAt: number): string {
  const elapsed = Math.floor((Date.now() - startedAt) / 1000);
  if (elapsed < 60) return `${elapsed}s`;
  if (elapsed < 3600) return `${Math.floor(elapsed / 60)}m ${elapsed % 60}s`;
  return `${Math.floor(elapsed / 3600)}h ${Math.floor((elapsed % 3600) / 60)}m`;
}
```

8. **Live ticking:** `useEffect` with `setInterval(1000)` while isOpen to force re-render for duration updates. Use a `tick` state counter. Clear interval on close.

9. **Keyboard navigation:**
   - Build flat list of agent rows (across all groups)
   - Track `selectedIndex` state (default 0)
   - ArrowDown: increment selectedIndex (wrap to 0)
   - ArrowUp: decrement selectedIndex (wrap to end)
   - Enter: jump to selected row's workspace. Use cross-store action:
     - Find which tab contains the agent's ptyId (traverse all tabs' pane trees)
     - Call `useTabsStore.getState().setActiveContext(tab.workspaceItemId, label)`
     - Call `useTabsStore.getState().switchTab(tab.id)`
     - Call onClose()
   - Esc: onClose()
   - Handle via `onKeyDown` on the Dialog element

10. **Agent-to-workspace mapping:**
    - Get all tabs from `useTabsStore`
    - For each agent, find which tab contains a leaf with matching ptyId
    - From tab.workspaceItemId, look up workspace name from `useWorkspaceStore`
    - Group agents by workspace name

11. **Wrap in react-aria `Dialog`** for focus trapping and aria attributes (same pattern as CreateModal).
