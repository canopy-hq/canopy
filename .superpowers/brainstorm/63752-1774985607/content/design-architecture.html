Design — Section 1: Overall Architecture

Project structure and module boundaries

Project Structure
superagent/
├── src-tauri/ ── Rust backend
│ ├── src/
│ │ ├── main.rs ── app entry, Tauri setup
│ │ ├── commands/ ── Tauri IPC command handlers
│ │ │ ├── pty.rs ── create/resize/write/kill PTY
│ │ │ ├── workspace.rs ── import repo, list, remove
│ │ │ ├── git.rs ── branches, worktrees, status
│ │ │ └── settings.rs ── app preferences CRUD
│ │ ├── pty/ ── PTY session management
│ │ │ ├── manager.rs ── session lifecycle, multiplexing
│ │ │ └── process_detect.rs ── agent running detection
│ │ ├── workspace/ ── workspace/repo management
│ │ │ ├── manager.rs ── repo registry, worktree ops
│ │ │ └── watcher.rs ── fs watcher for git state changes
│ │ └── state.rs ── shared app state (Tauri managed)
│ ├── Cargo.toml
│ └── tauri.conf.json
├── src/ ── SolidJS frontend
│ ├── App.tsx ── root layout
│ ├── components/
│ │ ├── sidebar/ ── workspace list, repo tree
│ │ ├── terminal/ ── xterm.js wrapper, PTY binding
│ │ ├── panes/ ── split pane manager
│ │ └── settings/ ── settings modal/page
│ ├── stores/ ── SolidJS reactive stores
│ │ ├── workspace.ts ── repos, branches, worktrees
│ │ ├── terminal.ts ── PTY sessions, agent status
│ │ └── settings.ts ── user preferences
│ ├── lib/
│ │ ├── tauri.ts ── typed Tauri command wrappers
│ │ └── keybindings.ts ── keyboard shortcut registry
│ └── styles/
│ └── tailwind.css
├── package.json
└── tailwind.config.ts

Key Rust Crates

portable-pty

Cross-platform PTY creation & management

git2

Git operations (branches, worktrees, status)

notify

Filesystem watcher for git state changes

sysinfo

Process tree inspection for agent detection

serde + serde_json

Serialization for IPC and config persistence

tokio

Async runtime for PTY I/O and fs watching
