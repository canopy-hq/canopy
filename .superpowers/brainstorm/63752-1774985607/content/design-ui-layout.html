Layout Components

Sidebar (220px)

Collapsible repo tree, branch/worktree list, import button. Resizable via drag.

Tab Bar

One tab per branch/worktree in active repo. Agent badge, worktree indicator, + button.

Pane Area

Recursive split panes. Each leaf = xterm.js terminal. Draggable splitters.

Status Bar

Repo name, branch, pane count, keyboard shortcut hints.
